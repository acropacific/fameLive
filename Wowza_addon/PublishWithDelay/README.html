Wowza Media Server Pro Examples: Publish With Delay

To install

  1. Copy the jar files in the included lib to the [install-dir]/lib/ folder.
  2. Create folder [install-dir]/applications/publishwithdelay.
  3. Create folder [install-dir]/conf/publishwithdelay and copy [install-dir]/conf/Application.xml into this new folder.
  4. Edit the newly copied Application.xml and make the following changes:

    • Set the Streams/StreamType to: live
    • Add the following <Module> definition as the last enter in the <Modules> list: 
      
<Module>
	<Name>ModulePublishWithDelay</Name>
	<Description>ModulePublishWithDelay</Description>
	<Class>com.wowza.wms.plugin.publishwithdelay.ModulePublishWithDelay</Class>
</Module>

To run

Double click client/publishwithdelay.html. Click Connect, click Publish, click first Play, click Start Delay and click second Play button. The first video object is the camera. The second is the non-delayed stream. The third is the delayed stream.

Note: The source code for this system is included in the src folder.
